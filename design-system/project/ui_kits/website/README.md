# Musketeers Website UI Kit

A high-fidelity, interactive recreation of the Musketeers marketing website. Built from the brand brief and visual identity direction — no existing codebase was available.

## Design Width
1440px desktop layout. Content max-width: 1200px. 64px horizontal padding.

## Screens
- **Home** — Hero (dark, full-bleed) + Services grid
- **Services** — 6-service grid with hover states
- **About** — Origin story + three brand pillars
- **Contact** — Contact info + form with send confirmation
- **Work** — Placeholder (portfolio TBD)

## Components
| File | Description |
|---|---|
| `Header.jsx` | Sticky top nav with logo, links, CTA, mobile menu |
| `Hero.jsx` | Full-bleed black hero with logomark, headline, pillar tags |
| `Services.jsx` | 3-column grid of 6 services, hover states |
| `Footer.jsx` | 4-column footer, dark background, legal row |

## Usage
Open `index.html` to view the interactive prototype. Navigation state persists via localStorage.

## Notes
- No production codebase was provided — this kit is built from brand guidelines only
- Portfolio/Work section is intentionally blank
- Mobile layout is partially handled (hamburger menu present); full responsiveness not implemented
